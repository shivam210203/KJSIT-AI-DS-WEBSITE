Avidya - Hardware Forensic Suite



image-souce: 123RF.com

The current forensic method mandates taking the suspect's PC into custody in order to extract useful evidence and proof. However, this method inevitably alerts the suspect, thereby making the process of providing justice more difficult. Introducing Avidya, a versatile tool that is agent-less and comes as a single binary executable that can be carried on a USB pen drive. It allows you to investigate the suspect's PC, providing disk, memory, and real-time network traffic capture. Avidya is a next-gen tool designed to catch the next-gen cybercriminals.

  • Conducting reconnaissance on the target.
  • Scanning the entire file system for information.
  • Retrieving specific files and directories.
  • Recovering deleted files from sectors.
  • Dumping RAM for analysis.
  • Capturing real-time network traffic.
  • Supporting multiple platforms.
  • Offering a single binary that doesn't require an agent.

  • Go programming language
  • gRPC
  • mTLS
  • Paseto tokens
  • Etcd as a distributed database
  • Kubernetes for high-availability and scalability
